This is a learning project to grasp the understanding of the Microservices and their interaction, the project becomes scalable with functionalities added at the later stage as we keep learning further.

# TravelBooking Microservices 

travel-booking-microservices/
│── eureka-server/           # Service Discovery
│── api-gateway/             # Gateway for Routing & Security
│── config-server/           # Centralized Configuration
│── user-service/            # Manages Users & Authentication
│── booking-service/         # Handles Travel Bookings
│── payment-service/         # Manages Payments
│── notification-service/    # Sends Email/SMS Notifications
